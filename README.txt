INSTALLATION
------------

 1) Copy the taxonomy_csv directory into the appropriate modules directory in
    your Drupal installation (recommended: sites/all/modules).

 2) Go to Administer > Site building > Modules and enable the
    Taxonomy CSV import module.

USAGE
-----

Go to Administer > Content management > Taxonomy and
click on the CSV import tab. Follow the instructions on that page.
